Error:

VirtualBox can't operate in VMX root mode. Please disable the KVM kernel extension, recompile your kernel and reboot (VERR_VMX_IN_VMX_ROOT_MODE)

artinya VirtualBox mendeteksi bahwa KVM (Kernel-based Virtual Machine) di Linux sedang aktif dan sudah “memegang” hardware virtualization (VMX), jadi VirtualBox tidak bisa memakainya.

Masalah ini umum terjadi kalau:

  1. Kamu menjalankan VirtualBox di dalam VM yang sudah pakai KVM (nested virtualization).
  2. Atau di host Linux, modul KVM (kvm_intel atau kvm_amd) sedang aktif dan menghalangi VirtualBox.

Cara mengatasinya di Linux Host

Unload modul KVM

sudo rmmod kvm_intel
sudo rmmod kvm_amd
sudo rmmod kvm

(Sesuaikan kvm_intel atau kvm_amd dengan CPU kamu — Intel atau AMD)

Cek apakah sudah hilang

lsmod | grep kvm

Jalankan VirtualBox lagi.

Biasanya ini langsung memperbaiki masalah.

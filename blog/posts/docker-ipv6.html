Docker IPv6? Kenapa?

docker-compose.yml

version: "3.7"

networks:
  v6net:
    driver: bridge
    enable_ipv6: true
    ipam:
      config:
        - subnet: "fd00:dead:beef::/64"

services:
  c1:
    image: alpine
    command: sh -c "sleep infinity"
    networks:
      v6net:
        ipv6_address: "fd00:dead:beef::11"

  c2:
    image: alpine
    command: sh -c "sleep infinity"
    networks:
      v6net:
        ipv6_address: "fd00:dead:beef::22"

Fire up!

docker-compose up -d

Enable IPv6 forwarding

user@ubuntu:~/ipv6-docker$ sudo sysctl -w net.ipv6.conf.all.forwarding=1
net.ipv6.conf.all.forwarding = 1

Pasang NAT66 pakai ip6tables

sudo ip6tables -t nat -A POSTROUTING -s fd00:dead:beef::/64 -o wlp6s0 -j MASQUERADE

-o wlp6s0 sesuai interface default kamu (Wi-Fi di kasus ini)

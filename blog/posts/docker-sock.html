Cannot connect to the Docker daemon at unix:///var/run/docker.sock. Is the docker daemon running?

Aduh, error apalagi ini? bego banget kan gue? oke jadi ini case pas gw nginstall docker di void linux, udah berhasil install gue coba docker pull hello-world muncul error gini Cannot connect to the Docker daemon at unix:///var/run/docker.sock. Is the docker daemon running?

Cara benerinnya gini:

Install docker

Bagi yang belum, install dulu dockernya

sudo xbps-install -S docker

Run docker daemon

sudo ln -s /etc/sv/docker /var/service/

Cek status docker

sudo sv status docker

Note: kalo masih ga jalan coba ketik perintah sudo sv up docker

Tambahin user ke group docker

sudo usermod -aG docker $(whoami)

Logout -> Login

Coba logout terus login lagi, abis itu ketik newgrp docker

Coba akses lagi

Ketik perintah ini, docker ps, hasilnya harusnya kek gini tot:

[febri@void ~]$ docker ps

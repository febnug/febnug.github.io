Fix grub dualboot Windows dan Linux

TL;DR

Ketika sudah selesai meng-install OS windows kita pengen dualboot sama linux tapi OS windowsnya ga muncul di grubnya linux?, begini cara benerinnya

Caranya cukup gampang, coba ketik ini dulu :

sudo update-grub

Nanti bakalan muncul error kaya gini

[febri@void ~]$ sudo update-grub
Generating grub configuration file ...
Found linux image: /boot/vmlinuz-6.3.12_1
Found initrd image: /boot/initramfs-6.3.12_1.img
Warning: os-prober will not be executed to detect other bootable partitions.
Systems on them will not be added to the GRUB boot configuration.
Check GRUB_DISABLE_OS_PROBER documentation entry.
done
[febri@void ~]$

Modif file /etc/default/grub, ketik perintah nano /etc/default/grub tambahin syntax ini :

GRUB_DISABLE_OS_PROBER="false"
GRUB_SAVEDEFAULT="false"

catatan : disini kasusnya syntax itu ga ada secara default di void linux, mungkin beberapa ada yang ada dan ada yang engga

Abis itu di save, ctrl + x abis itu y terus enter

Udah selesai update lagi grubnya

sudo update-grub

Nanti muncul kek gini, berarti berhasil

[febri@void ~]$ sudo update-grub
Generating grub configuration file ...
Found linux image: /boot/vmlinuz-6.3.12_1
Found initrd image: /boot/initramfs-6.3.12_1.img
Warning: os-prober will be executed to detect other bootable partitions.
Its output will be used to detect bootable binaries on them and create new boot entries.
Found Windows 7 on /dev/sda1
done
[febri@void ~]$

Reboot sistem, tinggal pilih mau pake linux apa windows :)

Konfigurasi IP Address (Netplan) di Ubuntu Server 24.04.1

Di kasus ini gw install ubuntu server 24.04.1 di virtualbox tapi pengen konek ke internet (bridged adapter) tapi pake DHCP, langsung aja caranya gini

Bikin file /etc/netplan/01-netcfg.yaml

network:
   version: 2
   renderer: networkd
   ethernets:
      enp0s3:
         dhcp4: true

Note: Di sesuaikan aja adapter network di ubuntunya, disini gw namanya enp0s3 caranya ketik perintah ip a

Di save, terus ubah permission filenya

sudo chown 600 /etc/netplan/01-netcfg.yaml

Terus ubah atau tambahkan nameserver google di /etc/resolv.conf

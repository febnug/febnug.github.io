Setting Jaringan

Pake iwctl, caranya, ketik iwctl

Ketika masuk shell iwctl :

[iwctl]# device list

Muncul interface devicenya, misal wlan0

Abis itu ketik :

[iwctl]# station wlan0 scan
[iwctl]# station wlan0 get-networks

Nanti muncul beberapa SSID wifinya, misal disini mau konek ke SSID Febri's Hotspot

Ketik :

[iwctl]# station wlan0 connect "Febri's Hotspot"

Tinggal masukin passwordnya, cek koneksi dengan ping

Partisi

Misal disk hdd atau ssd terbaca dengan nama /dev/sda

Tabelnya kaya gini :

Device		Boot	........    Size
----------------------------------------
/dev/sda1        *		    200M
/dev/sda2			    2G
/dev/sda3			    295G

Dengan catatan :

/dev/sda1 = bootloader
/dev/sda2 = swap
/dev/sda3 = /

Instalasi

Ketik perintah ini :

mkfs.ext4 /dev/sda1
mkswap /dev/sda2
swapon /dev/sda2
mkfs.btrfs /dev/sda3

disini gw pake filesystem btrfs

Abis itu edit file /etc/pacman.conf, ketik nano /etc/pacman.conf

Siglevel = Never
#LocalFileSigLevel = Optional
#RemoteFileSigLevel = Required

Ikutin aja kaya di atas, terus di save

Abis itu ketik :

timedatectl set-ntp true
mount /dev/sda3 /mnt
mkdir /mnt/boot
mount /dev/sda1 /mnt/boot

Kurang lebih kaya gini tampilan tree disknya, ketik lsblk

 sda
  |
  +--- sda1 .... /mnt/boot
  |
  +--- sda2 .... swap
  |
  +--- sda3 .... /mnt

Abis itu install base systemnya, ketik :

pacstrap /mnt base base-devel linux linux-firmware nano

Tunggu aja ampe selesai.

Kalo udah selesai, ketik :

genfstab /mnt
genfstab -U /mnt >> /mnt/etc/fstab
genfstab -U /mnt
arch-chroot /mnt /bin/bash

Pastiin lagi, edit file /etc/pacman.conf, ketik nano /etc/pacman.conf

Siglevel = Never
#LocalFileSigLevel = Optional
#RemoteFileSigLevel = Required

Abis itu di save

Lanjut, ketik :

pacman -S networkmanager grub
systemctl enable NetworkManager
grub-install /dev/sda
grub-mkconfig -o /boot/grub/grub.cfg

Abis itu ubah password root, ketik passwd input password baru

Ketik nano /etc/locale.gen, cari :

en_US.UTF-8 UTF-8
en_US ISO-88 59-1

Un-comment aja

Ketik locale-gen

Abis itu, ketik nano /etc/hostname, isi hostnamenya misal febri, terus di save

Ketik ln -sf /usr/share/zoneinfo/Asia/Jakarta /etc/localtime

Ketik exit

Unmount disk, ketik umount -R /mnt

Restart sistem, ketik reboot

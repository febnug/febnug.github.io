Upgrade offline di Ubuntu

Caranya cukup gampang, di linux cuma ketik :

apt-get upgrade --print-uris -qq | sed -n "s/'\([^ ]\+\)' \([^ ]\+\) \([^ ]\+\) MD5Sum:\([^ ]\+\)/wget -c \1/p" > upgrade.cmd
apt-get upgrade --print-uris -qq | sed -n -e "s/_[0-9]%[0-9a-f][0-9a-f]/_/" -e "s/'\([^ ]\+\)' \([^ ]\+\) \([^ ]\+\) MD5Sum:\([^ ]\+\)/\4 .\/\2/p" > md5sum.txt

di terminal

Copy file upgrade.cmd dan md5sum.txt ke flashdisk atau media penyimpanan yang laen

Di windows download dulu software di bawah ini terus di install

  1. wget
  2. md5sum

Masuk ke path direktori di cmd (tempat nyimpen file upgrade.cmd dan md5sum.txt

Ketik 

upgrade.cmd
terus Enter, tunggu ampe selesai

Abis itu ketik 

md5sum -c md5sum.txt
terus Enter, pastikan outputnya "ok" semua, kalo ngga, jalanin ulang upgrade.cmd nya

Masuk ke linux, copy semua file *.deb ke /var/cache/apt/archives/ caranya ketik

sudo cp *.deb /var/cache/apt/archives/

Kalo udah ketik 

apt-get upgrade

Tunggu ampe selesai

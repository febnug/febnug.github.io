About

FNOS is a cringe simple minimalist operating system written with B from scratch using x86 assembly language. It was written in order to learn about 8086 it quite likely will serve no use for you.

This hobby project is dedicated to me when studying OS development and I think I wasted too much of my time doing useless things, one of which was making this project. This project is also dedicated for you (anyone who views this webpage).

If you want to try my disgusting operating system, no don't try this.

How to use

You can run my disgusting operating system using USB stick then boot it with real hardware or using QEMU.

  1. Using real hardware, first you must download fnos.img then make a bootable USB stick using Rufus.
  2. Using QEMU, you can see at here.
